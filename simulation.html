IM31006 – Simulation

Credits: 3
Duration: One Semester (12–14 weeks)
Level: Undergraduate (Core)

Course Objectives

  • Components and concepts in Discrete event simulations
  • Design and implement simulation models using a programming language or simulation software.
  • Analyze the output of simulation models using statistical techniques.
  • Apply simulation to solve real-world problems in manufacturing, logistics, and service systems.
